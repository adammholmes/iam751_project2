iam751_project2
---------------
Solves a 2D heat equation using the Alternating Direction Implicit method (and the Peaceman-Rachford Scheme).

building
--------
```
autoreconf -i
./configure
make 
```

notes
-----
Use ***plot.sh*** to run the program and generate an mp4 video from the results.
